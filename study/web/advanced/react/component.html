# React 组件

# 无状态组件

所谓无状态组件就是指这个组件内只有一个 render 方法，没有其他任何的逻辑及状态。

无状态组件可以直接写成函数的形式，这个函数接受一个 props 返回一段 JSX。

为什么要写成无状态组件？因为无状态组件的性能要高，他只有一个函数，他不像其他的组件里面有生命周期和逻辑操作，会简单很多。
